### Qu'est-ce qu'un programme ?

Un ensemble d'instruction pouvant être executé par le hardware (en général le CPU).
Enregistré sur un disque dur. Disque dur = Mémoire non-volatile.
Contraire de la RAM = Mémoire volatile.

### Qu'est-ce qu'un processus
Programme chargé dans la mémoire vive pour être executé par un programme.

### La différences des deux.
#### Programme
Un programme execute les processus

#### Processus
PID = Identifiant de processus.
PPID = Identifiant du processus-parents.
Est executé par le programme.
Possède un contexte d'execution.

Un CPU est composé de plusieurs coeur.
Un coeur peut executer un seul processus.
Le processeur fait le lien avec la barette RAM.
Le processeur peut piocher les processus soit dans la mémoire vive dites (RAM) soit sur la mémoire statique dites dure (Disque dur).
